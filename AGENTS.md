This project's agent, AI-assistant, and contributor conventions are documented
in [`CLAUDE.md`](CLAUDE.md), which is the canonical source.

- [`CLAUDE.md`](CLAUDE.md) — Canonical project instructions, conventions, and
  workflow shortcuts.
- [`.windsurf/rules/`](.windsurf/rules/) — Detailed Windsurf rule files
  (coding, committing, documenting, etc.).

  procedures for common tasks.
