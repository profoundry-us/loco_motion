# LocoMotion Guides


These guides cover the broader philosophies and tooling that LocoMotion
recommends for building Rails applications. They are optional reading — you
can use the LocoMotion gem on its own by following the
[Installation section of the README](../../README.md#installation).

- [Getting Started](getting-started.md) — Stand up a new Dockerized Rails app
  with HAML, TailwindCSS, and DaisyUI.
- [Debugging & Testing](debugging.md) — Set up RSpec, Playwright, service
  objects, and the remote debugger.
- [Authentication & Error Handling](authentication.md) — Wire up OmniAuth, the
  Web Console, and BetterErrors.

For contributor and core-team documentation, see the
[developer guides](../dev_guides/).
